# Scientific Conference Management System

Web application for managing scientific conferences, submissions, reviews, and participants.

## Tech stack
- Django
- Django REST Framework
- React
- SQLite

## Features
- User authentication
- Conference creation and management
- Submission handling
- Admin panel

## Setup

1. Create virtual environment:
   python -m venv venv

2. Activate it:
   - Windows: venv\Scripts\activate
   - Linux/macOS: source venv/bin/activate

3. Install dependencies:
   pip install -r requirements.txt
